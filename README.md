# FinDash

FinDash is a beginner-focused financial literacy platform designed to help users
understand cash flow and make progress toward clear financial goals without
overwhelming complexity.

## Project Status
FinDash is in active development. This repository represents the redesigned,
architecture-first implementation.

## Design
System architecture and product rationale are documented here:
- [`docs/design.md`](docs/design.md)

## Core Focus
- Cash flow awareness
- Short-term actionable goals
- Long-term financial objectives
- Educational, goal-driven feedback

## Non-Goals (for now)
- Advanced investing analytics
- Power-user budgeting workflows
- Fully automated financial optimization
